# 鲜花销售管理系统 java836

## 项目概述

鲜花销售管理系统是一款基于前后端分离架构的管理系统，旨在为鲜花销售商提供全面的管理解决方案。系统采用 Spring Boot 和 Vue 技术栈开发，确保了高效性和稳定性。

## 技术栈

### 前端

- Vue
- CSS
- JavaScript
- ECharts
- Axios
- Element-UI

### 后端

- Java
- Spring Boot
- MyBatis-Plus
- MySQL
- JWT

## 功能模块

### 后台（管理员）

- 登录
- 首页
- 用户管理
- 商品管理
- 订单管理
- 财务管理
- 系统管理
- 充值管理
- 角色管理
- 个人中心

### 前台（用户）

- 注册
- 登录
- 首页
- 商城浏览
- 我的订单
- 我的收藏
- 个人中心
- 购物车
- 评论

## 系统角色

- 管理员（商家）：负责系统的管理和维护。
- 用户：浏览、购买鲜花并管理个人账户。

## 运行环境

- 开发工具：IDEA 或 Eclipse VSCode
- 数据库：MySQL

## 目录结构

```
-- 前端代码目录（Vue）
-- 后端代码目录（Spring Boot）
    |- src
    |   |- main
    |   |   |- java
    |   |   |- resources
    |   |   |- webapp
    |
    |- pom.xml
-- SQL数据库脚本
```

## 部署步骤

1. 准备所需的环境和依赖。
2. 创建数据库，执行 SQL 脚本。
3. 编译后端代码，打包成可执行 JAR 文件。
4. 部署前端代码到 Web 服务器。
5. 配置相关服务器和应用程序参数。
6. 运行后端服务。

## 核心亮点

- 基于成熟技术栈，保障系统稳定可靠。
- 前后端分离，易于维护和扩展。
- 完善的功能模块，满足鲜花销售业务需求。
